== Description ==

Start or grow your print-on-demand business with Print.com. This plug-in will make it easy for you to connect your WooCommerce store.

Key Features:
1. Easily set-up your Print.com account.
2. Connect your Products with a Print.com Product.
3. Manually or automaticly purchase the order.


== Installation ==

To allow this plug-in to work, you only need an account with Print.com.
Start by entering your account and password in the admin section of this plug-in.

**Don't have an account yet?**
You can request an account by contacting Print.com at +31 (0)88 751 11 11 or mailing to info@print.com.
